# Wedding Background Images

This folder contains background images for wedding card templates.

## Images to Add:

1. **golden-temple-golden-hour.jpg** - Golden Temple at golden hour
2. **golden-temple-soft-light.jpg** - Golden Temple with soft morning light
3. **bougainvillea-archway.jpg** - Archway with pink bougainvillea
4. **fortified-city.jpg** - Fortified city on hillside (Rajasthan)
5. **traditional-wedding-couple.jpg** - Traditional Indian wedding couple

## Instructions:

Please add the actual image files to this folder. The images should be:
- High resolution (at least 1920x1080)
- JPG or PNG format
- Optimized for web (compressed but good quality)

The templates will automatically use these images as backgrounds when selected.
